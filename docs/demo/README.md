- `frontend-dashboard.png`
- `api1-docs.png`
- `api2-docs.png`

These correspond to:

- running frontend at `http://localhost:5173`
- API 1 Swagger docs at `http://localhost:8001/docs`
- API 2 Swagger docs at `http://localhost:8002/docs`
